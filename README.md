# @ostai/next-css

Import `.css` files in your Next.js project. A @zeit/next-css fork to solve several anti-pattern issues.

## Install

```sh
$ npm i @ostai/next-css
```

## Usage

```js
const /next_css = require('@ostai/next-css')
```
